# 快速开始

使用包管理器yarn来为 JS-Finder 安装依赖项。

  • clone 这个仓库:
  git clone https://github.com/MisterBoole/JS-Finder.git
1
  • 进入项目并用 yarn安装:
  cd JS-Finder && yarn install
1
  • 在终端运行命令:
  yarn dev
1
  • 在浏览器打开:
  http://localhost:8080
1
