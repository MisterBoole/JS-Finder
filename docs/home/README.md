  ## Quick start

    Several quick start options are available:

    - Clone the repo: `https://github.com/MisterBoole/JS-Finder.git`
    - Install: `cd JS-Finder && yarn install`
    - Development run `yarn dev`
    - Open `http://localhost:8080` in your browser.
