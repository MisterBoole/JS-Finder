## 快速开始

使用包管理器[yarn](https://yarnpkg.com/)来为 JS-Finder 安装依赖项。

- clone 这个仓库:
```bash
  git clone https://github.com/MisterBoole/JS-Finder.git
```
- 进入项目并用 yarn安装:
```bash
  cd JS-Finder && yarn install
```
- 在终端运行命令:
```bash
  yarn dev
```
- 在浏览器打开:
```bash
  http://localhost:8080
```
